# Lab 01 — Basic Network Connectivity

## Objective

Establish communication between two hosts in the same network using a switch.

## Topology

![Topology](topology.png)

## IP Addressing

| Device | IP Address   | Subnet Mask   |
| ------ | ------------ | ------------- |
| PC0    | 192.168.1.10 | 255.255.255.0 |
| PC1    | 192.168.1.20 | 255.255.255.0 |

## Configuration

No switch configuration required. IP addresses were configured manually on end devices.

## Verification

Command used:
ping 192.168.1.20

Result:
Successful replies received.
