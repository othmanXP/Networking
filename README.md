# Network Engineering Labs

This repository documents my hands-on practice in networking, covering fundamental to advanced topics.

## Lab Categories

* Basic Networking
* Switching (VLANs, Trunking)
* Routing (Static, OSPF, etc.)
* Network Services (DHCP, NAT)
* Security (ACLs, Port Security)

## Labs

Labs are organized by topic and difficulty. Each lab includes:

* Objective
* Topology
* Configuration
* Verification
* Troubleshooting notes
